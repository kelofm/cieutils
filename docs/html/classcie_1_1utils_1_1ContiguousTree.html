void cie::utils::ContiguousTree< TNode, TIndex, TParallelTag >::visit ( const TFunctor &  rVisitFunctor) const

Invoke a functor on every node in the tree.

The callable functor should take a reference to a node and an index representing its depth. If the depth is not required, use begin and end to loop through the nodes in unspecified order instead.

Parameters
rVisitFunctorfunction to invoke on each node. It must take a node and level as argument, and produce a bool that indicates whether the visit loop should be continued.

Single thread version.
