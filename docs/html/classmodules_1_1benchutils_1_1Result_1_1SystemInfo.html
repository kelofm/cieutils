Reconstruct a SystemInfo instance from a dict or initialize one with the current system's info.

Only "OS", "hardware", and "python" are copied from inputDict and the rest of the entries are ignored. Missing entries are added using the current system's info. The internal dict has the following structure:

{
"OS" : {
"system" : "",
"release" : "",
"version" : ""
},
"hardware" : {
"machine" : "",
"processor" : "",
"memory" : ""
},
"python" : {
"build" : "",
"compiler" : "",
"version" : ""
}
}
Note
Fetching the size of the system's main memory requires the 'psutil' package.
Properties that could not be fetched are replaced by "unavailable".
