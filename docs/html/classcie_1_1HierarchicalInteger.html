static Level cie::HierarchicalInteger< Dim, TValue >::getLevel ( TValue  node)
staticnoexcept

Find which level node is located in.

Level boundaries:

  • Number of nodes in a cell: 2^D.
  • Number of cells in one level: 2^D^level.
  • Number of nodes in one level: 2^D^(level + 1)
  • Total number of nodes in a level and all levels below: 000001000..01000..01000..01000..0 | || || || | |–D–||–D–||–D–||–D–| | | |-----— level * D ----—|
