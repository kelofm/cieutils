static Ref<MPI> cie::mpi::MPISingleton::get ( )
static

Get the static MPI interface.

If the static MPI interface was not constructed and initialized yet, it will be after this function is called. If CiE was not compiled wiht MPI support, a serial dummy MPI interface is constructed.

Note
If you need to control what kind of MPI interface to construct, use MPISingleton::get(bool).

◆ get() [2/2]

static Ref<MPI> cie::mpi::MPISingleton::get ( Ptr< MPI_Comm >  p_comm)
static

Get the static MPI interface.

Parameters
p_commPtr to the MPI communicator to construct the interface with. If nullptr is passed, MPI_COMM_WORLD is duplicated if CiE was compiled with MPI support, otherwise a serial inter- face is constructed.

If the static MPI interface was not constructed and initialized yet, it will be after this function is called. The constructed interface is distributed if the input argument distributed is true, otherwise a serial dummy interface is created.

Exceptions
Ifthe static MPI interface already exist and p_comm is not its communicator.
