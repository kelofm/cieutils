Construct a parallel for object with thread local storage.

Template Parameters
TArgsparameter pack of thread local storage types.
Parameters
rPoolthread pool to use for execution.
rArgsinitializers for the thread local storages.

Each thread's local storage is initialized with the provided values, which is semantically equivalent to firstPrivate in OpenMP. The provided arguments are copied to each thread's local storage. Target functions must accept all arguments in matching order (after the index or container item).

Create a parallel for construct with initialized thread-local variables.

Each thread's local storage is initialized with the provided values, which is semantically equivalent to firstPrivate in OpenMP. The provided arguments are copied to each thread's local storage. Target functions must accept all arguments in matching order (after the index or container item).

Note
This is a factory function to help with reducing template clutter.
