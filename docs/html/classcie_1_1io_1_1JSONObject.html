Detailed Description

Basic interface for a json parser.

supported types for io:

  • bool
  • int
  • cie::Size
  • float
  • double
  • std::string
  • JSONObject
  • std::vector of any above except bool
  • StaticArray of any above with size 1, 2, or 3
